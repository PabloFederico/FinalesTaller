Practice problems on recursion & iteration

November 1, 2012 by HQN 2 Comments

  • Given a vector of integers, return the minimum integer
  • Given a vector of integers, return the product of all integers in the vector; 1 is returned if the vector is empty
  • Given a vector of integers, return a vector of integers which is the reverse of the input vector
  • Given a positive integer , return
  • Given two arrays of integers A and B return true if B is a subarray of A and false otherwise. (Imagine sliding B from left to right until you find a position where elements of B perfectly align with elements of A.)
